# Evidence

This directory contains generated evidence artifacts for Evidence-Driven Delivery.

Start here:

1. [evidence-summary.md](evidence-summary.md)
2. [release-decision.json](release-decision.json)
3. [traceability-report.md](traceability-report.md)
4. [traceability-matrix.json](traceability-matrix.json)
5. [test-report.json](test-report.json)
6. [junit.xml](junit.xml)

## Artifact Roles

| Artifact | Audience | Purpose |
|---|---|---|
| evidence-summary.md | Human | High-level release evidence overview |
| release-decision.json | Machine | Machine-readable release decision |
| traceability-report.md | Human | Human-readable traceability report |
| traceability-matrix.json | Machine | Machine-readable traceability matrix |
| test-report.json | Machine | Normalized test execution evidence |
| junit.xml | Machine | Raw pytest JUnit result |
